Biomass (general)
=======================================

This technology is a general switch that allows for a switching on and off of all the biomass technolgies.
By not deploying this technology, all the other advanced biomass technologies are deactivated.
This switch does not affect :ref:`wood_boiler`, :ref:`../wood_boiler`, wood_boiler_cp and wood_boiler_sg
