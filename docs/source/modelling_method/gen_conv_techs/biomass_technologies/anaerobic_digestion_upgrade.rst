Anaerobic Digestion Upgrade
=======================================

The anaerobic digestion upgrade describes a 
biomass technology that converts wet biomass
into biogas. This biogas can be used by any
gas-consuming entities in the district.

.. include:: ../../../how_to_use_the_model/input_csv_as_rst/anaerobic_digestion_upgrade.rst

The relationship between the in- and outflows is given by

.. math:: \mathtt{v\_gas\_agu} = \mathtt{efficiency} \cdot \mathtt{u\_wet\_bm\_agu}
